Getting Started

Thank you for purchasing Loop - Parallax HTML One-Page Template for Agency and Personal. This theme includes two templates that you can use for you or your client as required. Unlike the Agency type, Personal template has personality touch in term of design and content. By reading this documentation we hope you will get the most of its features. This template/theme is built with the latest Bootstrap 3 and SASS (with Compass). We recommend that you use SASS and Compass to customize the theme because it will be very easy since we have provided variables to change the color, font and others. If you have any questions that are beyond the scope of this help file, please feel free to contact us via email at contact@thedevelovers.com.

Directory Structure

After unzipping the downloaded file, you will get files structured as below:
  • documentation/: where this documentation is placed
  • themes/: theme files, includes
    • agency/: this template is intended for agency purposes.
    • personal/: this template is intended for personal purposes.

For both themes, the javascript files includes jQuery and the plugins are located at assets/js/. The main Javascript files used to initialize and configure the plugins are loop.js. So if you want to make some changes or additional codes, it should be at loop.js

General

Google Fonts

Google font used in this theme are Dosis and Open Sans. You can find the code on the head of each html files.

Icon Fonts

Most of the icons used are Font Awesome (http://fortawesome.github.io/Font-Awesome/icons/), but however we also use brankic icon like what we use in "Services" section. The font files are located at assets/fonts. So if you want to update Font Awesome icon, just replace the existing files.

Favicon

Replace all the favicons at assets/ico with corresponding size as provided on the demo. The number at file name, such as 57 means 57x57 pixels and so on. The favicon.png has dimension of 20x20 pixels.

Logo

Find the logo at assets/img directory named loop-logo.png. For Personal theme, it also has the white version loop-logo-white.png as the logo will change once adjusting navigation bar background. You can see the logo switch functions at loop.js as below.

Navigation

Find the navigation menu by finding id="main-nav", as you can see the hyperlink is pointed to the id of each content section such as #about to About section (id="about") and so on. The main-nav is very important for scrollspy to work, the active menu item will be determined by scrollspy as you scroll the page. The init code is located at loop.js as below.

We exclude Opera from scrollspy since it conflicts with full-screen background which need body on 100% height.

Footer

Footer includes copyright text and social icon list on Agency theme. If you don't want tooltip on each social icon please remove data-placement and data-toggle from the hyperlink tag. Please refer to http://fortawesome.github.io/Font-Awesome/icons/ for complete list of available social icons.
